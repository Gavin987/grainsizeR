# Code of Conduct

grainsizeR follows a simple contribution standard: be respectful, specific, and
evidence-focused. Keep discussions about code, documentation, data provenance,
and scientific methods constructive.

Harassment, personal attacks, or knowingly disruptive behavior are not
acceptable in issues, pull requests, or project discussions.

Project maintainers may edit, close, or remove contributions that are outside
the project scope or inconsistent with this standard.
