# Texture Polygon GitHub Backlog Draft

## Milestone: Official Texture Polygon Datasets

This backlog draft prepares future GitHub issues. It does not create issues,
tags, releases, package data, or polygon coordinates.

## Issue Templates by Workflow Stage

- Source manifest: identify and review primary official or academic sources.
- Evidence ledger: record source evidence and claims.
- Reconstruction: independently reconstruct coordinates from primary sources.
- Validation: validate geometry, boundaries, and classification examples.
- Documentation: document provenance, limitations, and user workflow.
- Review: confirm source, reconstruction, validation, and documentation status.
