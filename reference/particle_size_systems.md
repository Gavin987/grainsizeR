# List built-in particle-size boundary systems

`particle_size_systems()` returns source-aware metadata for
particle-size boundary systems used or planned by grainsizeR. These
entries describe particle-size boundaries only; they are not complete
texture classification systems and do not include texture-class polygon
coordinates.

## Usage

``` r
particle_size_systems()
```

## Value

A tibble describing particle-size boundary systems.
